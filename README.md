# GitHub Pages - Split MNIST Continual Learning

This folder contains a static GitHub Pages site for the Split MNIST continual-learning experiments.

## Local files

- `index.html` - main Hebrew RTL report page
- `styles.css` - site styling
- `assets/` - graphs, combined CSV, and Hebrew Word report

## Enable GitHub Pages

1. Push this repository to GitHub.
2. Open the repository on GitHub.
3. Go to `Settings -> Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Choose the branch, usually `main`.
6. Choose the `/docs` folder.
7. Save.

GitHub will publish the site after a short build. The published URL will appear on the same Pages settings screen.

## Notes

The page is fully static. It does not require Python, Conda, CUDA, or the original experiment outputs at runtime.
