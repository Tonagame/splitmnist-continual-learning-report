# Documentation Index

This folder contains the written documentation for the Split MNIST continual-learning project.

## Recommended Reading Order

1. `PROJECT_SCOPE.md` - what the project reproduces and what is outside scope.
2. `REPRODUCTION_REPORT.md` - main results and conclusions.
3. `IMPLEMENTATION_AUDIT.md` - honesty check: what is fully implemented, partial, or experimental.
4. `CODE_EXPLANATION.md` - how the implementation works.
5. `LSR_LITE_EXPLANATION.md` - the LSR-lite prototype and its ablations.
6. `PAPER_COMPARISON.md` - detailed comparison with the paper and reference code.
7. `AI_WORK_LOG.md` - summary of how AI assistance was used during the project.

## Assignment Support Files

- `takeaways.md` - reflective writing.
- `VIDEO.md` - short video checklist / placeholder.
- `METHODS_IMPLEMENTATION.md` - method-by-method implementation status.
